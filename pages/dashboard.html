📊 Dashboard Operacional

Este dashboard apresenta a análise de performance operacional com base em dados históricos, permitindo identificar padrões, gargalos e eficiência dos navios.

📌 Indicadores analisados

  • Atraso médio operacional
  • Relação entre carga e eficiência
  • Performance por navio
  • Evolução temporal dos indicadores
  • Identificação de outliers operacionais

💡 Insight

O projeto transforma dados operacionais em insights estratégicos para apoiar decisões e otimizar a eficiência logística.
